# Elevate-Labs-Internship-Task-8

Creating a basic interactive dashboard that shows sales performance by category, region, and month.

## Data Prep
- Imported dataset into Power BI.
- Cleaned and formatted date fields.
- Created custom DAX columns for:
- Month-Year (for timeline analysis)
- YearMonthSort (to maintain chronological order)
- Built interactive slicers for Category and Region.

## Dashboard Components
- Total Sales by Month-Year

Line chart showing sales trends from 2014 to 2017.
- Total Sales by Region

Bar chart comparing revenue across geographical regions.
- Total Sales by Category

Donut chart displaying category contribution toward total revenue.
- Filters: Category, Region

## Outcome

This dashboard enables users to quickly understand business performance and identify potential opportunities in high-performing regions and product categories.
